내 요구사항을 모두 적용해서 변경한 html, css, js파일 전문을 코드블록에 작성해주고 어디가 바꼈는지도 표시해줘
최대한 기존 코드를 유지하고 필요한 부분만 바꿔. 새로 탬플릿같은거? 띄우면 새로 쓰면서 덮어써지니까 꼭 평소에 쓰는 코드블록에 작성해줘. 아래 5가지가 내 요구사항이야
1. 파일을 올리면 압축하기 전에 파일 목록과 파일명 입력, 다운로드 박스를 띄워. 이름을 설정하고 클릭하는 다운로드 박스를 압축률 우선, 속도 우선 두가지 버튼을 만들고 클릭한 버튼에 맞는 모드로 압축을 수행하고 바로 다운로드해줘. 
2. 파일명 입력상자에 Result를 디폴트로 하고 전체선택 되어있어서 새로 입력하거나 지우기 한번만 눌러도 바로 사라지게 해줘. 그리고 파일 목록 오른쪽에 x를 추가해서 취소할 수 있고 추가로 이 상황에서 클릭을 하거나 새로 드래그를 하면 기존 파일들에 새 파일들을 추가할 수 있도록 해줘.
3. 폴더를 선택하면 실패하는걸 수정해줘. 그리고 그런 식으로 폴더 하나만 올리면 다운로드하는 파일명 디폴트를 그 올린 폴더명으로 해줘
4. 압축중일 때 뜨는 바가 늘어나는거를 개수를 고정하지 말고 바가 늘어나다가 보더에 닿으면 다음은 초기화하고 다시 늘어나는 식으로는 못하니? 못하면 20칸 채우면 초기화하는 식이어도 좋아.
5. 커버 이미지도 마우스 가져다대면 테두리 변하고 클릭해서 이미지 바꿀 수 있게 해줘
