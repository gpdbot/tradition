## 📄 README: 프로그램 개요 및 기능

### 전통놀이 개선판 (Traditional Game Improved Version)

| 항목 | 내용 |
| :--- | :--- |
| **프로그램 명** | 디시 전통놀이 개선판 |
| **개요** | 파일을 압축한 후, 커버 이미지(PNG)에 해당 압축 데이터를 결합하여 하나의 이미지 파일로 만드는 **스테가노그래피** 도구입니다. |
| **웹페이지** | https://gpdbot.github.io/tradition/ |

***

## ✨ 주요 기능 (Key Features)

* **다중 파일/폴더 입력:** 파일 선택 또는 드래그 앤 드롭을 통해 개별 파일 및 폴더 전체를 입력받아 압축할 수 있습니다.
* **커버 이미지 관리:** 커버 이미지를 선택하거나, 드래그 앤 드롭을 통해 새로운 이미지를 등록할 수 있습니다.
* **압축 모드 선택:** 저장 용량 우선 (`DEFLATE`) 또는 속도 우선 (`STORE`) 옵션을 선택하여 압축을 수행합니다.
* **파일명 지정:** 출력될 최종 PNG 파일의 이름을 지정할 수 있으며, 기본값은 **`Result.png`**입니다.

***

## 🖼️ 이미지 복원 방법 개요

본 프로그램이 생성한 파일은 이미지 데이터(PNG) 뒤에 압축된 파일 데이터(ZIP)가 붙어있는 구조입니다. 

따라서, 이 이미지 파일에서 숨겨진 데이터를 복원하는 과정은 매우 간단합니다.

1.  **파일 이름 변경 (Rename):**
    * 생성된 **`[파일명].png`** 파일을 **`[파일명].zip`**으로 이름을 변경합니다.
2.  **압축 해제 (Extraction):**
    * 이름이 변경된 `.zip` 파일을 일반적인 압축 해제 프로그램(예: 7zip, Bandizip 등)을 사용하여 내용을 추출합니다.

대부분의 압축 프로그램은 파일 끝에서 ZIP 아카이브의 메타데이터(EOCD 레코드)를 찾아 유효한 ZIP 파일로 인식하고 내부 파일을 정상적으로 해제합니다.
